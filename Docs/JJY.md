  - 기본적인 MySQL

1. mysql 로그인 명령어: mysql -u root -p~ (비밀번호)
2. 데이터베이스 보기: show databases;
3. 데이터베이스 전환: use (DB 이름);
4. 테이블 보기: show tables;
5. 테이블 안 데이터 보기: SELECT * FROM (테이블 이름);
